\subsection{Detailed Description}
The \hyperlink{a01183}{Program} is the main entrypoint for the X\+A\+CC A\+PI. Users with accelerator kernels must construct a valid \hyperlink{a01183}{Program} to be compiled and executed on the attached accelerator. Programs must be given the \hyperlink{a01111}{Accelerator} reference to be used and kernel source code at construction time.
