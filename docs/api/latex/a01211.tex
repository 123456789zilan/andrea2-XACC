\subsubsection*{template$<$typename Vertex, Graph\+Type type = Undirected$>$\newline
class xacc\+::\+Graph$<$ Vertex, type $>$}

The \hyperlink{a01211}{Graph} class provides a generic data structure modeling mathematical graph structures. It is templated on the vertex type, allowing for graphs with a wide variety of graph nodes (for example, in quantum computing -\/ graph of tensors, graph of Ising parameters, etc.)
