\subsection{Detailed Description}
The \hyperlink{a01127}{Compiler} class provides an extensible interface for injecting custom compilation mechanisms into the X\+A\+CC framework. Implementations provide a compile method that takes the kernel source code string, performs compiler-\/specific compilation mechanism, and returns a valid X\+A\+CC \hyperlink{a01175}{IR} instance modeling the result of the compilation.
