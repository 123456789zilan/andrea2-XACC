class xacc\+::\+Register\+Compiler$<$ T $>$}

\hyperlink{a01131}{Register\+Compiler} is a convenience class for registering custom derived \hyperlink{a01127}{Compiler} classes.

Creators of \hyperlink{a01127}{Compiler} subclasses create an instance of this class with their \hyperlink{a01127}{Compiler} subclass as the template parameter to register their \hyperlink{a01127}{Compiler} with X\+A\+CC. This instance must be created in the C\+PP implementation file for the \hyperlink{a01127}{Compiler} and at global scope.
