\subsection{Detailed Description}
The role of the \hyperlink{a00057}{C\+L\+I\+Parser} is to parse all command line options provided to an X\+A\+C\+C-\/enabled program. It takes upon construction to available argc and argv variables from the command line, and parses them to fill the \hyperlink{a00255}{Runtime\+Options} singleton and load any X\+A\+CC \hyperlink{a00059}{Compiler} or \hyperlink{a00030}{Accelerator} plugins.

It also queries all available Option\+Providers to display all available options to the X\+A\+CC user.
