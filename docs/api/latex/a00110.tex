\subsubsection*{template$<$typename T$>$\\*
class xacc\+::\+Register\+Preprocessor$<$ T $>$}

\hyperlink{a00110}{Register\+Preprocessor} is a convenience class for registering custom derived \hyperlink{a00094}{Preprocessor} classes.

Creators of \hyperlink{a00094}{Preprocessor} subclasses create an instance of this class with their \hyperlink{a00094}{Preprocessor} subclass as the template parameter to register their \hyperlink{a00094}{Preprocessor} with X\+A\+CC. This instance must be created in the C\+PP implementation file for the \hyperlink{a00094}{Preprocessor} and at global scope.
