class xacc\+::\+Register\+Accelerator$<$ T $>$}

\hyperlink{a01115}{Register\+Accelerator} is a convenience class for registering custom derived \hyperlink{a01111}{Accelerator} classes.

Creators of \hyperlink{a01111}{Accelerator} subclasses create an instance of this class with their \hyperlink{a01111}{Accelerator} subclass as the template parameter to register their \hyperlink{a01111}{Accelerator} with X\+A\+CC. This instance must be created in the C\+PP implementation file for the \hyperlink{a01111}{Accelerator} and at global scope.
