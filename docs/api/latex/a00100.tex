\subsection{Detailed Description}
The \hyperlink{a00100}{Quell\+E\+Graph} models an undirected weighted graph. It serves as the base class for all graph-\/related data structures in QuellE, such as adiabatic quantum computing problem inputs, hardware configurations, minor graph embeddings, and Ising parameters used as the machine-\/level instructions for an adiabatic quantum computer.

Its graph functionality is provided through delegation to the Boost distributed adjacency list, which also provides native parallelism for this graph.
