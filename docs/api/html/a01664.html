Detailed Description

template<typename T>
class boost::dll::experimental::imported_class< T >

This class represents an imported class.

Note
It must be constructed via boost::dll::import_class(const smart_library& lib, std::size_t, Args...)
Template Parameters
Thetype or type-alias of the imported class.
