Detailed Description

The KernelReplacementPreprocessor is a preprocessor for gate model quantum computing that analyzing quantum kernel source code and looks for occurrences of 'xacc::FUNCTION' strings (for example, xacc::QFT(qreg)).

Once one of these occurrences is found, this preprocessor queries the AlgorithmGenerator Registry for any available AlgorithmGenerators with that name. If such an AlgorithmGenerator exists, it generates the XACC IR instance for that algorithm, translates the IR to the Compiler's source code representation, and replaces the 'xacc::FUNCTION' occurrence with the new source code.
