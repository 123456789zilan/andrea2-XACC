Detailed Description

The IR interface is the base interface for derived accelerator-specific intermediate representations. At this level, an intermediate representation can be persisted to an assembly-like string, can be read in from file, and can be persisted to file. Since all XACC intermediate representations operate on an Accelerator Buffer, the IR interface also provides a setter for such a buffer.
