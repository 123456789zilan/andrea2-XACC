Detailed Description

template<typename ... InstructionParameter>
class xacc::quantum::ParameterizedGateInstruction< InstructionParameter >

The ParamaterizedGateInstruction is a GateInstruction that is templated on a list of variadic parameters that model the instructions gate parameters. For example, this class could be subclassed to provide a rotation gate with an angle theta (ParamaterizedGateInstruction<double>).
