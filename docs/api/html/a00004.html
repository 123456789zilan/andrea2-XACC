Building Prerequisites (Fedora 23)

I found that the latest CMake did not find Boost correctly. To fix this I downgraded to CMake 3.2 (rpm found here ftp://rpmfind.net/linux/fedora/linux/releases/22/Everything/x86_64/os/Packages/c/cmake-3.2.2-1.fc22.x86_64.rpm)

dnf remove cmake

wget ftp://rpmfind.net/linux/fedora/linux/releases/22/Everything/x86_64/os/Packages/c/cmake-3.2.2-1.fc22.x86_64.rpm

dnf install cmake-3.2.2-1.fc22.x86_64.rpm

dnf install mpich-devel boost-mpich-devel

Build QuellE

cd quelle && mkdir build && cd build cmake .. make
