Detailed Description

IEmbeddingAlgorithm is an interface that describes the functionality necessary to compute a minor graph embedding of one graph into another. Realizations of this interface should be able to indicate whether the given problem graph can even be embedded in the hardware graph, and should provide an implementation of computeEmbedding that executes their specific algorithm and returns an Embedding object. This Embedding object simply represents the problem-to-hardware vertex mapping.
