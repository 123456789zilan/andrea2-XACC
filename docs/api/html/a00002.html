Building QuellE Docker Image

To build the QuellE Docker image, simply execute the provided buildQuellEDocker.sh script with your code-int.ornl.gov GitLab Private Token passed as an argument:

./buildQuellEDocker TOKEN

You can find this token at https://code-int.ornl.gov/profile/account. Simply copy that token and paste it as the argument to the script.

Running the QuellE Docker Container

To create a new QuellE container, run the following:

docker run -it mccaskey/quelle bash

This will put your terminal into an interactive terminal session within the QuellE container. The tests and qcc QuellE Compiler are located in /quelle/build.

Running the QuellE Compiler with Docker

To run the quelleCompiler executable to compute the minor graph embedding of a problem graph into a hardware graph, execute the following:

docker run mccaskey/quelle /quelle/build/qcc –complete 5 –hName K44Bipartite –e ORNLCMRHeuristic

Or, for a more complicated example

docker run mccaskey/quelle /quelle/build/qcc –complete 20 –hName NasaChimera –e ORNLCMRHeuristic

Or, to run in parallel with MPI

docker run mccaskey/quelle mpiexec -np 2 /quelle/build/qcc –complete 20 –hName NasaChimera –e ORNLCMRHeuristic

In order to create a truly ephemeral container, ie deletes itself after execution, pass –rm to the run command:

docker run –rm mccaskey/quelle /quelle/build/qcc –complete 5 –hName K44Bipartite –e ORNLCMRHeuristic
