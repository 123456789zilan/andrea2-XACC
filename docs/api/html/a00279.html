Detailed Description

template<const int TotalNumberOfQubits>
class xacc::quantum::SimulatedQubits< TotalNumberOfQubits >

SimulatedQubits is an AcceleratorBuffer that models simulated qubits. As such, it keeps track of the state of the qubit buffer using a Rank 1 Fire Tensor.

It provides an interface for applying unitary operations on the qubit buffer state.
