Detailed Description

template<typename T, typename... TArgs>
class xacc::Registry< T, TArgs >

Registry is a Singleton that provides a mapping of string ids to creation functions that create and return the provided Registry template parameter T.

Clients can add new creation functions to be placed in the map with a unique name key, and can request that the Registry return a new created instance of the template parameter T.
