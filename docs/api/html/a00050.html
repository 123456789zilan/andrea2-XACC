Detailed Description

The Embedding class serves as a mapping of problem vertices to a set of vertices located in the hardware graph. Embeddings must be instantiated with the given problem and hardware graphs, and can indicate whether or not they are valid embeddings. It is up to clients and subclasses to construct the embedding mapping.
